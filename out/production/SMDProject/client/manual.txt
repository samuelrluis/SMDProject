usage: git [--version] [--help] [-C <path>] [-c name=value]
           [--exec-path[=<path>]] [--html-path] [--man-path] [--info-path]
           [-p | --paginate | --no-pager] [--no-replace-objects] [--bare]
           [--git-dir=<path>] [--work-tree=<path>] [--namespace=<name>]
           <command> [<args>]

These are common Client commands used in various situations:

connect to server
   register                 Regist new client in the Directory Service
   login			        Login into the Directory Service

finish connection
   logout			        Ends the connection with the server

work on the current directory
   copyfile			        Copy a file from server to client machine
   movefile			        Move a file from server to client machine
   getfilecontent	
   removefile		        Delete a file or an empty directory
   makedir                  Makes a directory on the current directory
   
directory commands
   changeworkingdirectory	Change current working directory for a new one
   getworkingdircontent	    Get info from the current directory content
   getworkingdirpath		Prints the current directory path
